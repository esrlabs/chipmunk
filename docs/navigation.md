We’ve put a lot of effort into making the `chipmunk` interface simple and intuitive, despite the complexity of the tasks it performs. Navigation focuses on fast access to recent work, favorite locations, and common actions.

### Recent sessions and favorites

Chipmunk keeps recent sessions available from the home screen. Favorite folders provide quick access to common log locations and include filtering by file name.
