At the heart of `chipmunk` lies its search functionality. Filters let you define reusable search conditions, keep them with a session, and apply them when needed.

### Managing filters

Filters can be created from the search input and adjusted later without recreating the session. They help keep relevant conditions available while exploring logs.

### Library

The library keeps the filters and charts applied to the current session in one place, so the active analysis setup stays easy to review and manage.

### Presets

Presets let you save filter and chart collections for reuse across sessions. They can also be imported and exported when you need to share or move an analysis setup.
