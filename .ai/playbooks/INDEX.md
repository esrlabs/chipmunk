# Chipmunk Playbooks Index

## Overview

Playbooks are procedural guides for recurring tasks that span one or more modules.
Use these when you need execution steps, sequencing, and validation criteria.

## Catalog

1. `.ai/playbooks/TEMPLATE.md`

## When to Add a Playbook

- The task is repeated frequently.
- The task crosses module boundaries.
- The task has a non-obvious validation or rollback path.

## How to Use

1. Copy `.ai/playbooks/TEMPLATE.md` into a new playbook file.
2. Fill in module-specific steps and checks.
3. Link the playbook from relevant knowledge files and from this index.
