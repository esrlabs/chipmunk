Basic settings
host Host the client should connect to.
port Port the client should connect to
username Username used to login.
password Password used to login.
timeout Sets the socket to timeout after the specified number of milliseconds of inactivity on the socket.
Extra settings
shellPrompt Shell prompt that the host is using.
loginPrompt Username/login prompt that the host is using.
passwordPrompt Password/login prompt that the host is using.
failedLoginMatch String or regex to match if your host provides login failure messages.
irs Input record separator. A separator used to distinguish between lines of the response.
ors Output record separator. A separator used to execute commands (break lines on input).
echoLines The number of lines used to cut off the response.
pageSeparator The pattern used (and removed from final output) for breaking the number of lines on output.
execTimeout, ms A timeout used to wait for a server reply when the 'exec' method is used.
sendTimeout, ms A timeout used to wait for a server reply when the 'send' method is used.
maxBufferLength, bytes Maximum buffer length in bytes which can be filled with response data.
initialLFCR Flag used to determine if an initial '\r\n' (CR+LF) should be sent when connected to server.
stripShellPrompt Whether shell prompt should be excluded from the results.
negotiationMandatory Disable telnet negotiations if needed.
debug Enable/disable debug logs on console.
