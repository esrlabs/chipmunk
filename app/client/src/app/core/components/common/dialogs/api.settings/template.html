LogViewer can work as standalone solution. You can load files from your file's system and parse it. But also you can connect to serial port. To do it, logviewer-service should be deployed on machine, which ports you want to listen. Here you can define address of this machine. Settings will be saved and restored with next using of LogViewer.
API Service Define address of API service.
Run-time service. Define address of run-time service (web socket server).
WS Protocol. Define protocol of web socket service. In most cases better to leave default value.
Reconnection timeout. Define reconnection timeout for web socket service.
Current state. Current connection state.
